## 1. 股票基础信息 (stock_basic)

| 字段 | 中文名 | 说明 |
|------|--------|------|
| ts_code | 证券代码 | 股票代码+交易所，如 000001.SZ |
| symbol | 股票代码 | 纯数字代码，如 000001 |
| name | 股票名称 | 公司简称 |
| area | 地区 | 注册地省份 |
| industry | 行业 | 所属行业 |
| fullname | 全称 | 公司全称 |
| market | 市场类型 | 主板/创业板/科创板 |
| list_date | 上市日期 | 首次上市日期 YYYYMMDD |
| is_hs | 沪深港通 | H=沪股通，S=深股通，N=否 |

## 2. 日线行情 (daily)

| 字段 | 中文名 | 说明 | 单位 |
|------|--------|------|------|
| ts_code | 证券代码 | 股票代码 | - |
| trade_date | 交易日期 | YYYYMMDD 格式 | - |
| **open** | 开盘价 | 当日第一笔成交价格 | 元 |
| **high** | 最高价 | 当日最高成交价格 | 元 |
| **low** | 最低价 | 当日最低成交价格 | 元 |
| **close** | 收盘价 | 当日最后一笔成交价格 | 元 |
| pre_close | 昨收价 | 前一交易日收盘价（复权后） | 元 |
| **change** | 涨跌额 | close - pre_close | 元 |
| **pct_chg** | 涨跌幅 | (change / pre_close) × 100 | % |
| **vol** | 成交量 | 当日总成交股数 | 手（1手=100股） |
| **amount** | 成交额 | 当日总成交金额 | 千元 |

> 💡 **投资提示**：收盘价是最重要的价格，用于计算涨跌幅、市值等。成交量放大通常意味着市场关注度提高。

## 3. 每日估值指标 (daily_basic)

### 3.1 估值指标

| 字段 | 中文名 | 说明 | 如何使用 |
|------|--------|------|----------|
| **pe** | 市盈率(静态) | 股价 / 去年全年每股收益 | 越低越便宜（但需结合增长看） |
| **pe_ttm** | 市盈率(动态) | 股价 / 近12个月每股收益 | 更能反映当前估值 |
| **pb** | 市净率 | 股价 / 每股净资产 | 低于1表示破净，可能被低估 |
| **ps** | 市销率(静态) | 总市值 / 去年营收 | 适用于亏损但有收入的公司 |
| **ps_ttm** | 市销率(动态) | 总市值 / 近12个月营收 | - |
| **dv_ratio** | 股息率 | 近12个月现金分红 / 股价 × 100 | 高股息策略关注此指标 |
| **dv_ttm** | 股息率(TTM) | 滚动12个月股息率 | % |

### 3.2 股本与市值

| 字段 | 中文名 | 说明 | 单位 |
|------|--------|------|------|
| **total_share** | 总股本 | 公司发行的全部股票数量 | 万股 |
| **float_share** | 流通股本 | 可在二级市场交易的股票数量 | 万股 |
| **free_share** | 自由流通股本 | 剔除限售股后的流通股 | 万股 |
| **total_mv** | 总市值 | 股价 × 总股本 | 万元 |
| **circ_mv** | 流通市值 | 股价 × 流通股本 | 万元 |

### 3.3 换手率

| 字段 | 中文名 | 说明 |
|------|--------|------|
| **turnover_rate** | 换手率 | 成交量 / 流通股本 × 100% |
| **turnover_rate_f** | 自由流通换手率 | 成交量 / 自由流通股本 × 100% |
| **volume_ratio** | 量比 | 当日成交量 / 过去5日平均成交量 |

> 💡 **投资提示**：
> - PE < 15 通常认为便宜，PE > 30 通常认为贵（需结合行业）
> - PB < 1 为"破净股"，银行股常见
> - 股息率 > 4% 可考虑作为高股息策略标的

## 4. 财务指标 (fina_indicator)

### 4.1 盈利能力指标

| 字段 | 中文名 | 说明 | 优秀标准 |
|------|--------|------|----------|
| **roe** | 净资产收益率 | 净利润 / 净资产 × 100 | > 15% 优秀 |
| **roe_dt** | 扣非净资产收益率 | 扣除非经常性损益后的 ROE | > 15% 优秀 |
| **roa** | 总资产收益率 | 净利润 / 总资产 × 100 | > 10% 优秀 |
| **grossprofit_margin** | 毛利率 | (营收 - 成本) / 营收 × 100 | > 30% 较好 |
| **netprofit_margin** | 净利率 | 净利润 / 营收 × 100 | > 10% 较好 |
| **op_of_gr** | 营业利润率 | 营业利润 / 营收 × 100 | - |

### 4.2 成长能力指标

| 字段 | 中文名 | 说明 |
|------|--------|------|
| **tr_yoy** | 营收同比增长率 | (本期营收 - 去年同期) / 去年同期 × 100% |
| **op_yoy** | 营业利润同比增长率 | 营业利润的年增长率 |
| **netprofit_yoy** | 净利润同比增长率 | 净利润的年增长率 |
| **dt_netprofit_yoy** | 扣非净利润同比增长率 | 更真实反映主业盈利增长 |

### 4.3 运营效率指标

| 字段 | 中文名 | 说明 |
|------|--------|------|
| **ar_turn** | 应收账款周转率 | 营收 / 平均应收账款，越高越好 |
| **inv_turn** | 存货周转率 | 成本 / 平均存货，越高越好 |
| **assets_turn** | 总资产周转率 | 营收 / 平均总资产 |
| **current_ratio** | 流动比率 | 流动资产 / 流动负债，> 2 安全 |
| **quick_ratio** | 速动比率 | (流动资产-存货) / 流动负债，> 1 安全 |

### 4.4 财务健康指标

| 字段 | 中文名 | 说明 | 安全标准 |
|------|--------|------|----------|
| **debt_to_assets** | 资产负债率 | 总负债 / 总资产 × 100 | < 60% 一般安全 |
| **equity_ratio** | 权益乘数 | 总资产 / 净资产 | - |
| **ocf_to_netprofit** | 经营现金流/净利润 | 现金流质量指标 | > 1 表示利润质量好 |

> 💡 **核心指标**：
> - **ROE**（净资产收益率）是巴菲特最看重的指标，反映股东资金的使用效率
> - **毛利率**反映产品竞争力
> - **资产负债率**反映财务风险

## 5. 利润表 (income)

利润表展示公司在一段时期内的经营成果（赚了多少钱）。

### 5.1 收入类

| 字段 | 中文名 | 说明 |
|------|--------|------|
| **total_revenue** | 营业总收入 | 公司所有业务收入总和 |
| **revenue** | 营业收入 | 主营业务收入 |
| **int_income** | 利息收入 | 银行类公司的核心收入 |
| **prem_earned** | 已赚保费 | 保险公司收入 |
| **comm_income** | 手续费及佣金收入 | 券商、银行等 |

### 5.2 成本费用类

| 字段 | 中文名 | 说明 |
|------|--------|------|
| **total_cogs** | 营业总成本 | 所有成本费用总和 |
| **oper_cost** | 营业成本 | 与收入直接相关的成本 |
| **sell_exp** | 销售费用 | 广告费、销售人员工资等 |
| **admin_exp** | 管理费用 | 行政管理相关费用 |
| **fin_exp** | 财务费用 | 利息支出、汇兑损失等 |
| **rd_exp** | 研发费用 | 科技公司关键指标 |

### 5.3 利润类

| 字段 | 中文名 | 说明 |
|------|--------|------|
| **operate_profit** | 营业利润 | 主营业务的利润 |
| **total_profit** | 利润总额 | 税前利润 |
| **income_tax** | 所得税 | 企业所得税 |
| **n_income** | 净利润 | 税后利润，归属于股东 |
| **n_income_attr_p** | 归母净利润 | 归属于母公司股东的净利润 |

> 💡 **利润表阅读顺序**：
> 营业收入 → 减去成本 → 毛利润 → 减去三费 → 营业利润 → 加减营业外 → 利润总额 → 减去税 → 净利润

## 6. 资产负债表 (balancesheet)

资产负债表展示公司在某一时点的财务状况（家底有多少）。

**核心公式**：资产 = 负债 + 所有者权益

### 6.1 流动资产（一年内可变现）

| 字段 | 中文名 | 说明 |
|------|--------|------|
| **money_cap** | 货币资金 | 现金及银行存款 |
| **notes_receiv** | 应收票据 | 银行承兑汇票、商业汇票 |
| **accounts_receiv** | 应收账款 | 客户欠款（关注账龄） |
| **prepayment** | 预付款项 | 预付给供应商的钱 |
| **inventories** | 存货 | 原材料、在产品、产成品 |
| **oth_cur_assets** | 其他流动资产 | 理财产品等 |
| **total_cur_assets** | 流动资产合计 | - |

### 6.2 非流动资产（长期资产）

| 字段 | 中文名 | 说明 |
|------|--------|------|
| **lt_rec** | 长期应收款 | 分期收款形成的长期债权 |
| **lt_eqt_invest** | 长期股权投资 | 对子公司/联营公司投资 |
| **fix_assets** | 固定资产 | 厂房、设备、机器 |
| **cip** | 在建工程 | 正在建设中的项目 |
| **intan_assets** | 无形资产 | 专利、商标、土地使用权 |
| **goodwill** | 商誉 | 并购溢价（风险点！） |
| **total_nca** | 非流动资产合计 | - |
| **total_assets** | 资产总计 | 流动 + 非流动 |

### 6.3 流动负债（一年内要还）

| 字段 | 中文名 | 说明 |
|------|--------|------|
| **st_borr** | 短期借款 | 一年内到期的银行贷款 |
| **notes_payable** | 应付票据 | 开出的银行/商业汇票 |
| **acct_payable** | 应付账款 | 欠供应商的钱 |
| **adv_receipts** | 预收款项 | 客户预付的定金 |
| **payroll_payable** | 应付职工薪酬 | 欠员工的工资 |
| **taxes_payable** | 应交税费 | 欠税务局的税 |
| **total_cur_liab** | 流动负债合计 | - |

### 6.4 非流动负债

| 字段 | 中文名 | 说明 |
|------|--------|------|
| **lt_borr** | 长期借款 | 一年以上银行贷款 |
| **bond_payable** | 应付债券 | 发行的公司债 |
| **total_ncl** | 非流动负债合计 | - |
| **total_liab** | 负债合计 | 流动 + 非流动 |

### 6.5 所有者权益（净资产）

| 字段 | 中文名 | 说明 |
|------|--------|------|
| **total_hldr_eqy_exc_min_int** | 归母所有者权益 | 归属母公司股东的净资产 |
| **minority_int** | 少数股东权益 | 子公司中不属于母公司的部分 |
| **total_hldr_eqy_inc_min_int** | 所有者权益合计 | 净资产总额 |

> ⚠️ **风险提示**：
> - 商誉过高 → 并购风险，可能爆雷
> - 应收账款过高 → 回款困难，可能坏账
> - 存货过高 → 可能滞销，需要减值

## 7. 现金流量表 (cashflow)

现金流量表展示公司现金的流入流出（钱从哪来、花到哪去）。

**现金流比利润更真实**，因为利润可以调节，但现金骗不了人。

### 7.1 经营活动现金流（主业造血能力）

| 字段 | 中文名 | 说明 |
|------|--------|------|
| **c_fr_sale_sg** | 销售商品收到的现金 | 主营业务收入的现金 |
| **c_fr_oth_operate_a** | 收到其他经营活动现金 | 押金、罚款等 |
| **c_paid_for_goods_s** | 购买商品支付的现金 | 采购支出 |
| **c_paid_to_for_empl** | 支付给职工的现金 | 工资等 |
| **c_paid_for_taxes** | 支付的各项税费 | 税款 |
| **n_cashflow_act** | 经营活动现金流净额 | 收入 - 支出，**核心指标** |

### 7.2 投资活动现金流（对外投资）

| 字段 | 中文名 | 说明 |
|------|--------|------|
| **c_recp_disp_fix_assets** | 处置固定资产收到现金 | 卖设备、厂房收入 |
| **c_pay_for_fix_assets** | 购建固定资产支付现金 | 买设备、建厂房支出 |
| **c_pay_for_invest_ls** | 投资支付的现金 | 对外投资支出 |
| **n_cashflow_inv_act** | 投资活动现金流净额 | 通常为负（在扩张） |

### 7.3 筹资活动现金流（融资活动）

| 字段 | 中文名 | 说明 |
|------|--------|------|
| **c_fr_borr** | 取得借款收到的现金 | 银行贷款 |
| **c_pay_to_repa_borr** | 偿还债务支付的现金 | 还贷款 |
| **c_pay_for_dvd_int** | 分配股利支付的现金 | 分红支出 |
| **n_cash_flows_fnc_act** | 筹资活动现金流净额 | - |

### 7.4 现金净变化

| 字段 | 中文名 | 说明 |
|------|--------|------|
| **n_incr_cash_cash_equ** | 现金净增加额 | 三大活动现金流之和 |
| **c_cash_equ_beg_period** | 期初现金余额 | 年初有多少现金 |
| **c_cash_equ_end_period** | 期末现金余额 | 年末有多少现金 |

> 💡 **现金流分析技巧**：
> - 经营现金流 > 净利润 → 利润质量高
> - 经营现金流持续为负 → 主业不赚钱，危险信号
> - 投资现金流为负 + 经营现金流为正 → 健康扩张

## 附录：常用财务指标计算公式

### 估值类
```
市盈率(PE) = 股价 / 每股收益 = 总市值 / 净利润
市净率(PB) = 股价 / 每股净资产 = 总市值 / 净资产
市销率(PS) = 股价 / 每股营收 = 总市值 / 营业收入
股息率 = 每股股息 / 股价 × 100%
```

### 盈利类
```
毛利率 = (营收 - 成本) / 营收 × 100%
净利率 = 净利润 / 营收 × 100%
ROE = 净利润 / 净资产 × 100%  (杜邦分析核心)
ROA = 净利润 / 总资产 × 100%
```

### 杜邦分析
```
ROE = 净利率 × 资产周转率 × 权益乘数
    = (净利润/营收) × (营收/总资产) × (总资产/净资产)
```

### 偿债类
```
资产负债率 = 总负债 / 总资产 × 100%
流动比率 = 流动资产 / 流动负债 (> 2 安全)
速动比率 = (流动资产 - 存货) / 流动负债 (> 1 安全)
```
